# cyHelp
Designed for the CyberPeace Peace-a-thon 2022 with [Tharunya](https://github.com/Tharunya07/)

### Uses
1. Flask
2. BeautifulSoup
3. Smokesignal
4. [face-recognition](https://pypi.org/project/face-recognition/)
